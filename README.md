# Sudoku Solver in C++

A simple **Sudoku Solver** built in C++ using the **Backtracking Algorithm**. It solves a 9x9 Sudoku puzzle by filling in missing values while following all Sudoku rules.

---

## Features

- Solves 9x9 Sudoku puzzles  
- Uses backtracking and recursion  
- Validates rows, columns, and 3x3 grids  
- Console-based output  

---

## How to Run

```bash
g++ -o sudoku_solver main.cpp
./sudoku_solver
```

---

## Concepts Used

- Backtracking  
- Recursion  
- Arrays
